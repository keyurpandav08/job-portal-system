# CareerLink Asset System

## Rules
- All assets must support Light and Dark themes
- Prefer SVG over PNG/JPG
- No hard-coded colors in shared assets
- Use CSS variables or `currentColor`

## Folder Usage
- light/: Light theme specific assets
- dark/: Dark theme specific assets
- shared/: Theme-adaptive assets

## Do Not
- Do not use stock images without customization
- Do not add white-only or dark-only icons
- Do not mix assets directly inside components
